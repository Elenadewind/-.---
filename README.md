## LegalDataProcessor v0.1.0: исправление алгоритма с ИИ‑поддержкой


## О проекте
Модуль обрабатывает юридические документы (договоры, претензии), выявляет ключевые сущности и проверяет валидность. В этой версии исправлен критический баг в логике классификации, с использованием подсказок ИИ‑ассистента.


## Цель доработки
Модуль автоматизирует анализ юр. документов (договоры, претензии): извлекает ключевые сущности, проверяет валидность и сигнализирует об ошибках. Версия с исправленным багом в классификации — для надёжной интеграции в бизнес‑процессы.
Повысить точность валидации до 95 %+ на тестовой выборке из 200 документов.

## Что исправлено
- Баг: неверная трактовка сроков действия пунктов (затрагивал 30 % договоров).
Проблема: пропуск обязательных полей (обнаружен в 15 % кейсов).


## Как использовать
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите тест: `python test_processor.py`
3. Подключите как модуль в свой проект.
4. python
from processor import LegalProcessor
proc = LegalProcessor()
result = proc.analyze("договор.txt")
print(result.issues)  # список ошибок
